# Quizet 🎯  
A simple Laravel-based quiz application that fetches questions from the [Open Trivia Database](https://opentdb.com) API. Users can select a category, answer multiple-choice questions, and view their final score.

---

## 🚀 Features

- Category selection for quiz topics  
- Multiple-choice questions with random options  
- Score tracking and result page  
- Responsive UI with Bootstrap  
- Fully containerized using Docker  
- Deployed on [<img src="https://img.shields.io/badge/Render-46E3B7?style=flat&logo=render&logoColor=black" height="25">](https://quizet-eq8v.onrender.com/)
